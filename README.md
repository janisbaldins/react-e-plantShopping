# e-plantShopping

e-plantShopping is a React-based shopping application that simulates an online plant store.  
The app allows users to browse products, add items to a cart, manage quantities, and view the total order value.

This project demonstrates modern frontend development practices including:
- React component architecture
- State management
- Shopping cart logic
- Clean UI structure

The application is designed as a learning project and showcases how an e-commerce flow can be implemented using React.
